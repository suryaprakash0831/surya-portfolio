## 🛠️ How to Host on GitHub

1. **Create a new repo** on GitHub named `surya-portfolio`
2. **Push this folder:**
   ```bash
   git init
   git add .
   git commit -m "🎢 Launch: Surya Prakash S Portfolio"
   git remote add origin https://github.com/YOUR_USERNAME/surya-portfolio.git
   git push -u origin main
   ```
3. **Enable GitHub Pages:**
   - Go to repo → Settings → Pages
   - Source: **GitHub Actions**
4. Your site will be live at `https://YOUR_USERNAME.github.io/surya-portfolio`
