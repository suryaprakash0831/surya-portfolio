# Quick Deployment Guide 🚀

## Upload to GitHub in 5 Minutes

### Step-by-Step Instructions:

1. **Go to GitHub.com**
   - Sign in to your account
   - Click the "+" icon (top right)
   - Select "New repository"

2. **Create Repository**
   - Repository name: `portfolio` (or your choice)
   - Description: "My Professional Portfolio Website"
   - Select "Public"
   - **DO NOT** check "Add a README file"
   - Click "Create repository"

3. **Upload Your Files**
   - On the quick setup page, click "uploading an existing file"
   - Open the `surya-portfolio` folder on your computer
   - Select ALL files and drag them into the GitHub page
   - **Make sure you include:**
     - index.html
     - styles.css
     - script.js
     - README.md
     - images folder (with both projects and sequence subfolders)
   - Add commit message: "Initial portfolio upload"
   - Click "Commit changes"

4. **Enable GitHub Pages**
   - Click "Settings" tab (top of repository)
   - Scroll down and click "Pages" in left sidebar
   - Under "Source":
     - Branch: Select "main"
     - Folder: Select "/ (root)"
   - Click "Save"
   - Wait 2-3 minutes

5. **View Your Site**
   - Your website will be live at:
   - `https://[your-github-username].github.io/portfolio`
   - Example: `https://suryaprakash0831.github.io/portfolio`

## Troubleshooting

**Images not showing?**
- Make sure the `images` folder was uploaded with both subfolders
- Check that file paths are correct (case-sensitive)

**Site not loading?**
- Wait a few minutes after enabling GitHub Pages
- Check that `index.html` is in the root directory
- Refresh your browser cache (Ctrl+Shift+R or Cmd+Shift+R)

**Want to update content?**
- Edit files on GitHub directly, or
- Upload new versions of the files

## Custom Domain (Optional)

To use your own domain (e.g., www.suryaprakash.com):
1. Buy a domain from GoDaddy, Namecheap, etc.
2. In GitHub Pages settings, add your custom domain
3. Update your domain's DNS settings to point to GitHub Pages
